## Relevant data för filmer
* TMDB id
* Titel (+ original titel för internationella filmer)
* Regissör
* Året som den släpptes
* Affisch

## Relevant data för serier
* TMDB id
* Titel
* Startår / hur länge det pågått
* Affisch

## Relevant data för skådespelare och regissörer
* TMDB id
* För- och efternamn
* Födelsedag / ålder
* Bild / headshot
